Padlockd - Ultra-secure note-taking application

A Full Stack Secure Notes Application built using Spring Boot, ReactJS, and Tailwind CSS, demonstrating advanced security concepts including JWT, OAuth2, multi-factor authentication, and secure password practices. This project implements cutting-edge security features to ensure robust protection for user data, integrating frontend and backend seamlessly with Spring Security 6.
