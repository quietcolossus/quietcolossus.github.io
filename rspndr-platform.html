RSPNDR: Alarm Response Marketplace

RSPNDR is a dynamic, Uber-like alarm response marketplace connecting alarm monitoring companies with nearby licensed security guards. Through a streamlined, responsive interface, RSPNDR enables efficient dispatch and response tracking for alarms at residential and commercial locations.

The platform includes multiple Angular front-end applications, native Android & iOS apps, and a robust Java Spring Boot back end deployed on AWS.

How RSPNDR Works

  1. Alarm Triggered
    When an alarm is activated, monitoring companies are instantly notified and can dispatch a guard with a single click.
  2. Alarm Dispatched to RSPNDR Network
    The alarm enters the RSPNDR marketplace, where it is assigned to the nearest responder through RSPNDR's mobile app.
  3. Responder Accepts the Alarm
    The closest available responder accepts the alarm, and their progress is visible in real-time through the RSPNDR Portal.
  4. Responder Completes Investigation and Submits Report
    A detailed incident report, with photos and timestamps, is sent via the mobile app, available immediately for monitoring company review.
  5. Site Secured
    The incident is resolved, and site security is reestablished.

Key Features

  • Real-Time Alarm Dispatch and Tracking - Integrated monitoring with instant guard dispatch and live status updates via the RSPNDR Portal.
  • Detailed Incident Reporting - Secure and detailed reports with images and timestamps, available in real-time for monitoring companies.
  • User-Friendly Guard Interface - Mobile apps on Android and iOS provide a smooth experience for guards to manage responses and submit reports.
  • Admin and Management Tools - Angular front-end tools designed for managing users, incidents, and real-time reporting.

Front End (Angular)

  • Angular (TypeScript) - Responsive and efficient UIs for both client-facing and internal applications.
  • RxJS - Reactive programming for managing asynchronous data streams.
  • NgRx - State management to handle complex data flow in large-scale applications.
  • SCSS - Modular and customizable styling for component-based design.
  • Angular Material - UI component library for rapid, consistent, and accessible UI development.

Back End (Java Spring Boot)

  • Spring Boot - REST APIs and business logic, supporting link-shortening, Twilio integration, and authentication.
  • Spring Security - Comprehensive security framework for authentication and authorization.
  • Spring Data JPA - Data access abstraction with streamlined CRUD operations and custom queries.
  • AWS (EC2, S3, RDS) - Scalable cloud infrastructure for application deployment, data storage, and database management.
  • Docker - Containerization for environment consistency and streamlined deployment.
  • Redis - Caching layer for performance optimization on frequently accessed data.

DevOps and Deployment

  • CI/CD Automation - Fully automated pipelines through GitHub Actions for smooth deployment and continuous integration.
  • Authentication and Security - Robust, secure user authentication and real-time data access with Keycloak JWT and role-based access.
